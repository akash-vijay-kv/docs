title: "Netra SDK"
---

Netra is a comprehensive Python library for AI application observability that provides OpenTelemetry-based monitoring, and tracing for LLM applications. It enables easy instrumentation, session tracking, and privacy-focused data analysis for AI systems.

## Key Features

<CardGroup cols={2}>
  <Card title="Comprehensive AI Observability" icon="magnifying-glass">

  </Card>
  <Card title="Privacy Protection" icon="shield">
    Advanced PII detection and masking with multiple detection engines
  </Card>
  <Card title="Security Scanning" icon="lock">
    Prompt injection detection and prevention to secure your AI applications
  </Card>
  <Card title="OpenTelemetry Integration" icon="chart-line">
    Industry-standard tracing and metrics for seamless observability
  </Card>
</CardGroup>

<CardGroup cols={2}>

  <Card title="Session Management" icon="users">
